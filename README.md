# SysManage
## 系统管理模块

#### 简介： 
一个简单的后台管理系统,主要功能包括：
- 用户管理
- 菜单管理
- 角色管理
- 菜单功能管理

1. 后台框架：SpringMVC MyBatis 
2. 数据库：MySql 
3. 前台：基于Bootstarp的superUi组件框架
